This is a model for prediction the diabetes of a patient. We need some data of patient like bmi,age etc to predict sugar level in body.
Be sure to use the same version of the code as the version of the docs you're reading. You probably want the latest tagged version, but the default Git version is the master branch.

step-1: Clone the repository

step-2: Open shell or any command line and move to your project directory

step-3: And install all required packages on your virtual enviroment by "$ pip install -e ."

step-4: Run flask server "$ flask run"

step-5: You could run test_request.py for testing your project or you could use third party application i.e."Postman" and make a post request.
